skill.search

Search facade: local keyword search + remote backend dispatch.

skill.search.backends: Registry[SkillSource] = Registry('backends', keys=[])

Registry of remote skill backends (e.g., github, local directories).

skill.search.search(query: str, *, max_results: int = 10, local_only: bool = False, backends: list[str] | None = None) → list[SkillInfo][source]

Search for skills across local index and remote backends.

With local_only=True, only searches locally installed skills. Otherwise, searches both local and remote, with local results first.

>>> isinstance(search('test', local_only=True, max_results=0), list)
True
