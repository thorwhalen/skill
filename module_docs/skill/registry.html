skill.registry

Generic plugin registry with entry point discovery.

class skill.registry.Registry(name: str, *, entry_point_group: str | None = None)[source]

A name-keyed registry of typed objects with lazy entry point discovery.

Each Registry wraps a dict and can auto-discover plugins registered via importlib.metadata entry points under the group skill.<registry_name>.

>>> r = Registry('demo')
>>> r['foo'] = 42
>>> r['foo']
42
>>> list(r)
['foo']
>>> len(r)
1
>>> del r['foo']
>>> len(r)
0
property name: str

Registry name (also used to derive the entry point group).

>>> Registry('agents').name
'agents'
